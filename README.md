# Full Stack Job Portal

## Project Overview
A full-stack job portal where recruiters can post jobs and job seekers can apply.

## User Roles
- Admin
- Recruiter
- Job Seeker

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB / PostgreSQL
- Authentication: JWT
